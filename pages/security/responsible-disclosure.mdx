# Responsible Disclosure of Security Vulnerabilities

At Langfuse, we value the security community and prioritize system security. We encourage the responsible disclosure of security vulnerabilities to help us protect the security and privacy of our users.

## Reporting a Vulnerability

If you discover a security vulnerability in Langfuse, please send actionable vulnerability reports to **security@langfuse.com**.

We commit to:

- Acknowledging receipt of your vulnerability report in a timely manner
- Providing an estimated timeframe for addressing the vulnerability
- Notifying you when the vulnerability has been fixed
- Recognizing your contribution (with your permission)

## Guidelines for Responsible Disclosure

When reporting vulnerabilities, please:

1. Provide detailed information about the vulnerability, including steps to reproduce
2. Allow reasonable time for us to address the issue before public disclosure
3. Avoid accessing, modifying, or deleting data without permission
4. Act in good faith and do not exploit the vulnerability beyond what is necessary to demonstrate the issue

## Recognition

Please note that we currently do not operate a bug bounty program. However, we do recognize security researchers who have helped improve Langfuse's security.
