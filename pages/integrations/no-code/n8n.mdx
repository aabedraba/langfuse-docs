# Integration: n8n

n8n is a workflow automation tool that allows you to connect to various services and automate your workflows.

## Prompt Management Integration

Langfuse [Prompt Management](/docs/prompts/get-started) is available via an n8n node.

Learn more in the [n8n Node documentation](/docs/prompts/n8n-node).

## Tracing Integration

There is popular demand for a Langfuse Tracing integration for n8n. Please upvote the related GitHub discussion [here](https://github.com/orgs/langfuse/discussions/4397).
