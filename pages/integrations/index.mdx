# Integrations

Langfuse is designed to be the most open and flexible platform for LLM engineering that integrates with all the major LLM providers, frameworks, and tools.

Langfuse Tracing can be integrated on the:

- **Application-level (recommended)**
- Gateway-level integration
- As an OpenTelemetry backend
- In no-code agent builders

You can always use the [public API](/docs/public-api) which allows to build your own integration.
