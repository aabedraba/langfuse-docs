What's new in Gemini 2.5 Pro Experimental? From [Google release notes](https://blog.google/technology/google-deepmind/gemini-model-thinking-updates-march-2025/):

- Gemini 2.5 Pro Experimental: Google's most advanced model for complex tasks, leading on LMArena leaderboard and showing strong reasoning and code capabilities
- Enhanced reasoning capabilities: State-of-the-art performance across benchmarks requiring advanced reasoning, including math and science
- Native multimodality with long context: Ships with 1M token context window (2M coming soon), capable of comprehending text, audio, images, video, and entire code repositories

Try Gemini 2.5 Pro Experimental in the [Langfuse LLM playground](/docs/playground) today.
